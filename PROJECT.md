# hunyuan2++

HiCache++ — DMD/Prony EXPONENTIAL velocity forecasting (new method) on Hunyuan3D-2 mini, with HiCache (Hermite) as the comparison baseline. The DMD basis is exact on the exponential feature-ODE solution class where the polynomial drifts, so it stays lossless at larger skip intervals. See `hicache-plus-plus`.
